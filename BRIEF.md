# My Links - Project Brief
A personal link-in-bio page. One page, clean design, mobile first 
I want to send people here from my social profiles

## Tech
Pure HTML, CSS, and JavaScript - no frameworks, no build tools 
Single index.html file (CSS and JS can be seperate files)
Deployes to Vercel as a static site

## Design 
Centered card layout, max-width 480px
My name as a heading, the sub headers Product Strategist, Mother, Sourdough Enthusist, Delinquient Yogi underneath. Make Product Strategist the main subheader and make the others more whimsical text. 
Profile photo placeholder (circle, 120px)
Light mode by default, optoion to toggle to dark 
Smooth hover over the animations on link buttons
Use a serif font for the main text and then a whimsical text for the subheader that matches well
